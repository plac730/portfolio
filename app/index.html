Blog App

GithubLive Site

Front-End: HTML, CSS, Javascript, Semantic UI

Back-End: Node, Express, MongoDB, Mongoose

Description: A blog site demonstrating an implementation of RESTful routes as a standardized convention of handling CRUD operations. Blog entries that can be created, edited, and deleted are stored in a Mongo Database.

Travel Site

GithubLive Site

Front-End: HTML, CSS, Javascript

Description: A mock travel site for a travel agency. The site is mobile-responsive and utilizes several modern web development tools such as gulp, babel, postcss, and webpack. The CSS is written according to the BEM methodology in mind, allowing the code to be easily scaled and maintained in the future.

Movie Finder

GithubLive Site

Front-End: HTML, CSS

Back-End: Node, Express, OMDB API

Description: A movie search web app that makes use of the OMDB API. Enter one or more words and the top 10 matching results from the Open Movie Database will be returned along with the movie date and movie poster, if there is one.

Open Face Chinese Poker

Github
